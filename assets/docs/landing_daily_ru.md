## Твоя ежедневная карта

Примерно на четвёртом-пятом раскладе символический язык начинает работать. Смартфон перестаёт быть источником отвлечения и становится надёжным якорем.

### Что предлагает приложение

- **Ежедневный ритуал** — Управляемый расклад С·П·Р из трёх карт — одна сессия в день. Осознанно, не вызывает зависимости.
- **Дневник размышлений** — Прошлые расклады сохраняются и доступны для просмотра. Наблюдай, как паттерны психики становятся видны за недели.
- **72 состояния Лилы** — Каждая карта — одно из 72 состояний сознания древней ведической системы. Не предсказание, а принудительная ясность.
- **Философия медленных технологий** — Один ритуал в день. Без бесконечной ленты. Глубокая интеграция до следующего озарения.

> *«Наполовину древняя оракульная система, наполовину утренний ритуал — 2000-летняя ведическая система на смартфоне, созданная для обычного утра вторника, когда тревога приходит раньше тебя».*

*LeelaClue — инструмент для саморефлексии. Приложение не делает предсказаний и не даёт профессиональных советов. Оно предоставляет зеркало — смелость смотреть в него ты приносишь сам.*
